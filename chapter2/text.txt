ชนิดข้อมูล
1.Boolean
2.Numeric เก็บค่าตัวเลข 2 กลุ่มใหญ่ๆ
2.1. Int (Integer) ตัวเลขจำนวนเต็มไม่เก็บติดลบ
2.2. Float ตัวเลขทศนิยม
3. String ตัวอักษรหรือชุดข้อมูล

Control Structure คือรูปแบบการเขียนโปรแกรม มี 3 แบบ ใหญ่
1.Sequential = ทำงานแบบบนลงล่าง
2.Selection (หรือ Condition) = การสร้างทางเลือกจากเงือนไขที่กำหนด เช่น if, if-else, switch
3.Iteration (หรือ Loop) = การสร้าง block สำหรับการทำคำสั่งซ้ำตามเงื่อนไขที่กำหนด ซึ่งปกติจะประกอบไปด้วย for while, do-while
