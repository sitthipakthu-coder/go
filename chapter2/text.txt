Control Structure คือรูปแบบการเขียนโปรแกรม มี 3 แบบ ใหญ่
1.Sequential = ทำงานแบบบนลงล่าง
2.Selection (หรือ Condition) = การสร้างทางเลือกจากเงือนไขที่กำหนด เช่น if, if-else, switch
3.Iteration (หรือ Loop) = การสร้าง block สำหรับการทำคำสั่งซ้ำตามเงื่อนไขที่กำหนด ซึ่งปกติจะประกอบไปด้วย for while, do-while

Data structure
รูปแบบข้อมูบอื่นๆที่สามารเก็บได้
1.Aray = รูปแบบการเก็บข้อมูลเป็น sequence หลาบข้อมูลในตัวแปรเดียวกันไว้ เช่น var a [5]Int
2.Slice = คล้ายๆ Aray แต่อนุญาตให้เปลี่ยนขนาดได้(Aray ต้องระบุขนาดเสมอ) เช่น var s []Int
3.Map = ข้อมูลคล้ายๆ dictionnary สามารถเก็บ key คู่กับ value ตรงๆได้เช่น map[string]Int
